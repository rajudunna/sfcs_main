EXTRA SHIPMENT ACHIEVEMENT

Actual Achievement % = (((Cut Quantity / Customer Order Quantity)*100)-100) - Loss%.

Production Loss % = (Input Quantity - Output Quantity)/Input Quantity * 100.

Target Achievement % = (Customer Order Quantity / Committed Ship Qty) * 100.

Extra Shipment Achievement Completed Schedules Count = Actual Achievement % > Target Achievement %

Extra Shipment Achievement Color Indication=Actual Achievement % / Target Achievement %.


EXTRA SHIPMENT ACHIEVEMENT COLOR INDICATION
0% - 25% RED
26% - 60% ORANGE
61% - 85% YELLOW
ABOVE > 85% GREEN
